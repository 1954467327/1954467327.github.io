# 错误码规范
由于业务系统错误码定义不一，也无法区别业务错误和技术错误
## 格式定义
错误码长度为8位，组成格式为1+2+2+3的方式。
1. 第一位长度为1代表错误码为业务类还是技术类，用E表示业务，S表示技术。
2. 第二位长度为2，代表系统标识，此处本系统暂填写固定值LT
3. 第三位长度为2，代表错误码的小类，分为业务类和技术类，具体参考附录中的错误码三级分类表。
4. 第四位长度为3，代表具体小类编号，企架规范可为数字或字母，本系统使用001~999来定义，如超出再使用字母+数字编号。
