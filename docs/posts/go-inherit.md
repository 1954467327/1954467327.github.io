# Go语言的继承(组合/嵌入)

当多个结构体存在相同的属性(字段)和⽅法时,可以从这些结构体中抽象出结构体,在该结构体中定义这些相同的属性和⽅法，其它的结构体不需要重新定义这些属性和⽅法，只需嵌套⼀个匿名结构体即可。也就是说:在Golang中，如果⼀个struct嵌套了另⼀个匿名结构体，那么这个结构体可以直接访问匿名结构体的字段和⽅法，从⽽实现了继承特性。1

## 继承语法说明

1. 结构体可以使⽤嵌套匿名结构体所有的字段和⽅法，即:⾸字⺟⼤写或者⼩写的字段、⽅法，都可以使⽤。
2. 匿名结构体字段访问可以简化。如：cat.Age --->cat对应的结构体中找是否有Age字段，如果有直接使⽤，如果没有就去找嵌⼊的结构体类型中的Age

```go
cat: &Cat{}
cat.Animal.Age = 3
// 简化
cat.Age = 3
```

3. 当结构体和匿名结构体有相同的字段或者⽅法时，编译器采⽤就近访问原则访问，如希望访问匿名结构体的字段和⽅法，可以通过匿名结构体名来区分。

```go
// 如果你仍然想调用“父类”的实现，可以用 s.person.SetAge(age)。
func (s *Student) SetAge(age int) error {
	if age < 6 || age > 30 {
		return fmt.Errorf("学生年龄不合法: %d", age)
	}
	return s.person.SetAge(age)
}
```

4. Golang中⽀持多继承：如⼀个结构体嵌套了多个匿名结构体，那么该结构体可以直接访问嵌套的匿名结构体的字段和⽅法，从⽽实现了多重继承。为了保证代码的简洁性，建议⼤家尽量不使⽤多重继承，很多语⾔就将多重继承去除了，但是Go中保留了。
5. 嵌⼊的匿名结构体有相同的字段名或者⽅法名，则在访问时，需要通过匿名结构体类型名来区分。c.Cat.age, c.Dog.age;
6. 结构体的匿名字段可以是基本数据类型。如：int 用于作为从逻辑上定义的万能值，在某个场景含义是A，在另一个场景含义是B。
7. 嵌套匿名结构体后，也可以在创建结构体变量(实例)时，直接指定各个匿名结构体字段的值。匿名变量初始化；
8. 嵌⼊匿名结构体的指针也是可以的。
9. 结构体的字段可以是结构体类型的。（组合模式）
