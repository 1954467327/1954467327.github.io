启动介绍

简洁至上

采用以 Markdown 为核心的项目结构进行最小化设置，能帮助您专注于写作。

Vue 引擎驱动

享受 Vue 的开发体验，使用 Markdown 中的 Vue 组件，并利用 Vue 开发自定义主题。

性能

VuePress 会为每个页面生成预渲染的静态 HTML 文件，并且在页面加载完成后会以单页应用程序的形式运行。

主题

提供了一个默认主题，无需额外设置即可使用。您还可以选择社区提供的主题，或者自行创建一个主题。

插件

灵活的插件接口，使插件能够为您的网站提供大量即插即用的功能。

构建者

默认的打包工具是 Vite，不过 Webpack 也受支持。选择你喜欢的那个！
